Dual-Color Capsule — Logical Defects

Visual Reference Guide · Print Quality Control

10 Defect Types
REF
✓ Normal / Acceptable
Two distinct colors correctly aligned with the capsule seam. Clean boundary, full coverage on both halves.
DEF 01
Color Swap / Reversal
Cap and body colors are interchanged. The expected color positions are reversed from specification.
Cause: Incorrect recipe/template loaded, or capsule fed upside-down.
DEF 02
Misregistration
The color boundary doesn't align with the physical capsule seam. One color overlaps into the other half.
Cause: Sensor timing offset or mechanical misalignment.
DEF 03
Missing Print (Half)
One color zone is completely absent. The capsule appears as a single color or shows the base capsule shell color.
Cause: Ink starvation, clogged nozzle, or broken print head.
DEF 04
Color Overlap / Bleed
The two colors overlap at the seam, creating a visible blended/mixed zone that is neither color.
Cause: Band width parameter set too wide; ink not dry before rotation.
DEF 05
Skewed / Tilted Print
The print boundary runs diagonally rather than perpendicular to the capsule axis, creating a slanted color division.
Cause: Capsule holder wear, vibration, or rotation speed inconsistency.
DEF 06
Incomplete Coverage
One color zone is only partially printed, leaving an uncolored gap between the two color zones.
Cause: Partial nozzle blockage or low ink pressure.
DEF 07
Ghosting / Double Impression
A faint secondary print appears offset from the main print, creating a visible shadow or echo effect.
Cause: Print roller bounce, double feed, or residual ink on drum.
DEF 08
Orientation Error (Inverted)
The capsule is fed inverted (180°). Colors appear on the correct zones but body/cap positions are swapped relative to assembly.
Cause: Capsule orientation sensor failure or hopper jam.
DEF 09
Incorrect Band Width
The color zone is narrower (or wider) than specified. The color boundary does not reach or exceed the capsule midpoint.
Cause: Band width parameter misconfiguration.
DEF 10
Repeat / Wrap-around Error
The color pattern repeats on the same capsule — a second color band appears in the wrong zone due to roller sync failure.
Cause: Encoder/roller synchronization error; capsule slippage.
